CRANKSHAFT POSITION (CKP) SENSOR REMOVAL/INSTALLATION [ZJ, Z6]

B3E014018220W01

Caution
• When replacing the CKP sensor, make sure there is no foreign material on it such as metal shavings. If it is installed with foreign material, the sensor output signal will malfunction resulting from fluctuation in magnetic flux and cause a deterioration in engine control.

1. Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION [ZJ, Z6].)

2. Disconnect the negative battery cable.

3. Perform the following procedure for easier access.

(1) Remove the under cover.

4. Remove the CKP sensor.

5. Install in the reverse order of removal.

CKP sensor tightening torque
7.8-10.8 N·m {80-110 kgf·cm, 70-94 in·lbf}
