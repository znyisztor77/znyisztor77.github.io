REAR HEATED OXYGEN SENSOR (HO2S) REMOVAL/INSTALLATION [ZJ, Z6]

B3E014018860W07

1. Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION [ZJ, Z6].)

2. Disconnect the negative battery cable.

3. Perform the following procedure for easier access.

European (L.H.D.) specs.
1. Remove the air cleaner cover. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [ZJ, Z6].)
U.K. specs.
1. Set the air cleaner cover out of the way. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [ZJ, Z6].)

4. Install the SST to the rear HO2S.

5. Remove the rear HO2S.

6. Install in the reverse order of removal.

Front HO2S tightening torque
29-49 N·m {3.0-4.9 kgf·m, 22-35 ft·lbf}
