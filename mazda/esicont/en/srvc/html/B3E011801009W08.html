SPARK PLUG REMOVAL/INSTALLATION [LF]

B3E011801009W08

Caution
• If a spark plug that is not as specified is installed, sealing performance will be deteriorated. Install only the specified spark plug when replacing.

1. Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION [LF].)

2. Disconnect the negative battery cable.

3. Remove the plug hole plate. (See PLUG HOLE PLATE REMOVAL/INSTALLATION [LF].)

4. Remove the ignition coil. (See IGNITION COIL REMOVAL/INSTALLATION [LF].)

5. Remove the spark plugs using a plug-wrench.

6. Install in the reverse order of removal.

Tightening torque
10-14 N·m {1.1-1.4 kgf·m, 8-10 ft·lbf}
