DTC P0112

IAT sensor circuit low input

DETECTION CONDITION

• The PCM monitors the IAT sensor signal at PCM terminal 1AH. If the PCM detects the IAT sensor voltage below 0.16 V, the PCM determines that the IAT sensor circuit has malfunction.
Diagnostic support note
• This is a continuous monitor (CCM).
• The MIL illuminates if the PCM detects the above malfunction condition during first drive cycle.
• PENDING CODE is available if the PCM detects the above malfunction condition.
• FREEZE FRAME DATA is available.
• DTC is stored in the PCM memory.

POSSIBLE CAUSE

• IAT sensor malfunction
• Short to ground in wiring harness between MAF/IAT sensor terminal D and PCM terminal 1AH
• Short each wiring harness IAT signal circuit and IAT ground circuit
• PCM malfunction
