KNOCK SENSOR (KS) REMOVAL/INSTALLATION [ZJ, Z6]

B3E014018920W03

1. Remove the battery cover. (See BATTERY REMOVAL/INSTALLATION [ZJ, Z6].)

2. Disconnect the negative battery cable.

3. Remove the intake manifold. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [ZJ, Z6].)

4. Install the SST to the KS.

5. Remove the KS.

6. Install in the reverse order of removal.

KS tightening torque
19.6-34.3 N·m {2.0-3.4 kgf·m, 14.5-24.5 ft·lbf}
