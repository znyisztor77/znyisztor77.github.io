SUPPLIER IDENTIFICATION PROCEDURE

B3E090266900W02

Note
• The supplier can vary with the module. When asking the supplier (service center) for repair or replacement, identify the supplier and fill in the Audio Repair Order Form with the following procedures.

Identification Using the Label or Inscribed Lettering

1. Remove the audio unit.

2. Verify the supplier by referring to the label attached to each unit.

Identification Using the On-board Diagnostic Test Mode.

1. Start the on-board diagnostic test mode.

2. Identify the device and supplier codes by referring to the LCD display.

Note
• If no DTC is stored, no codes will be displayed.
