REAR OIL SEAL REPLACEMENT [ZJ, Z6]

B3E011011399W01

1. Remove the following parts:

ATX

1. Automatic transaxle (See AUTOMATIC TRANSAXLE REMOVAL/INSTALLATION [Z6].)
2. Drive plate (See DRIVE PLATE REMOVAL/INSTALLATION.)

MTX

1. Clutch release cylinder (See CLUTCH RELEASE CYLINDER REMOVAL/INSTALLATION.)
2. Manual transaxle (See MANUAL TRANSAXLE REMOVAL/INSTALLATION [F35M-R].)
3. Flywheel (See CLUTCH UNIT REMOVAL/INSTALLATION.)

2. Remove the oil seal lip using a cutting knife.

3. Remove the oil seal using a flathead screwdriver with the tip protected by a rag to prevent crankshaft oil seal damage.

4. Apply clean engine oil to a new rear oil seal.

5. Insert the oil seal into the cylinder block by hand.

6. Install the rear oil seal using the SST.

Compression amount
From the cylinder block side end
0-0.5 mm {0-0.019 in)

7. Install the following parts:

ATX

1. Drive plate (See DRIVE PLATE REMOVAL/INSTALLATION.)
2. Automatic transaxle (See AUTOMATIC TRANSAXLE REMOVAL/INSTALLATION [Z6].)

MTX

1. Flywheel (See CLUTCH UNIT REMOVAL/INSTALLATION.)
2. Manual transaxle (See MANUAL TRANSAXLE REMOVAL/INSTALLATION [F35M-R].)
3. Clutch release cylinder (See CLUTCH RELEASE CYLINDER REMOVAL/INSTALLATION.)
