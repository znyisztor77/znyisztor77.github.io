SOLAR RADIATION SENSOR REMOVAL/INSTALLATION

B3E074061751W01

1. Disconnect the negative battery cable.

2. Pry the solar radiation sensor from the dashboard using a flathead screwdriver wrapped with protective tape.

3. Remove in the order indicated in the table.

Caution
• After the solar radiation sensor removal, the sensor connector could fall in the dashboard making the installation difficult. Therefore, hold the rooted end of the sensor connector using a clip or similar tool to prevent it from falling.

1
Solar radiation sensor connector
2
Solar radiation sensor

4. Install in the reverse order of removal.
